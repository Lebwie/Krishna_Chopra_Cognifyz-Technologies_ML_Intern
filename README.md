# 🧠 Cognifyz Tech Task 1 - Regression Models

This repository contains my solution for **Task 1** of the Cognifyz internship program. The goal of the project is to implement and compare various regression algorithms to predict a target variable from a given dataset.

---

## 📁 Project Structure

```
cognifyz-task-1/
├── COGNIFYZ_TECH_TASK_1.ipynb
├── [dataset.csv]
├── requirements.txt
├── README.md
├── LICENSE
```

---

## 🚀 Algorithms Implemented

The following machine learning regression models were used in this project:

* Linear Regression
* Lasso Regression
* Ridge Regression
* ElasticNet
* K-Nearest Neighbors Regressor
* Decision Tree Regressor
* Random Forest Regressor
* Gradient Boosting Regressor

---

## 📊 Dataset

The dataset used in this task is provided in the `data/` folder. It consists of features used to train and test different regression models.
